# Flask simple application

Run the Python script app.py by typing

```console
python app.py
```

Open the web site in the browser by clicking on "Open in Browser" in the notification that pops up when running the flask application.

Change the parameters in the url, e.g. `https://animated-space-yodel-7x55jqpvvvhp4vp-5000.app.github.dev/hello/Peter`
